โครงสร้างไฟล์
```
startupDaliydoApp
├─ App.js
├─ app.json
├─ assets
│  ├─ adaptive-icon.png
│  ├─ favicon.png
│  ├─ icon.png
│  └─ splash-icon.png
├─ index.js
├─ package-lock.json
├─ package.json
└─ src
   ├─ backend
   └─ frontend 
```

เมื่อ clone ไปแล้วให้ npm install 

## Tech Stack

### Frontend (Mobile App)
* **Framework:** React Native (Expo)
* **Emulator:** Android Studio (Pixel 9 Pro)
* **Language:** JavaScript

### Backend (Server & Database)
* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MySQL
* **Tools:** Postman (API Testing)
